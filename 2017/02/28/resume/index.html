工作经历

华兴资本 （ 2016年3月 ~ 今 ）

Arrow 财物顾问系统

华兴资本的 Arrow 财物顾问系统是给公司内部的 Advisors 用的系统，他们的业务流程由线下迁移到线上，并在系统中规范化管理，与此同时，系统中也维护了较全面的投资机构和创业公司等创投数据。这个项目是去年 3 月份我入职之后开始立项，项目前端只有我一个人负责，包括技术选型，代码的从 0 到 1，部署，以及后续的维护和优化。项目的前端技术栈主要有，AngularJS,jQuery,Ajax,Grunt,SASS 等。通过这个项目，进一步增加了项目从无到有的构建已经部署运维的实战经验，也在这个项目中不断踩坑，不断成长。这篇博客 就是记录了其中一个解决问题的过程。

华兴官网

华兴官网之前是由一个外包团队做的，华兴成立深圳 IT 团队后，华兴官网的维护工作就交给我们团队做了，我主要是帮忙修改维护一下网站内容。当时用的框架是 ThinkPHP。

Coding ( 2014年11月 ~ 2016年3月)

coding.net 主站

coding.net 是一个国内最大的一站式云端开发平台，秉承着“让开发更简单”的使命，将代码托管、项目管理、WebIDE 等开发工具集成到浏览器中，免除繁杂的开发环境部署，降低开发和部署软件的成本，提升团队协作沟通效率，提高软件开发生产力。我在这个项目中参与了前后端的日常开发和维护工作，技术栈主要有：后端 Java，Spring MVC 框架，MySQL 数据库，前端的技术栈主要是 AngularJS。

码市

码市 是 coding 旗下产品，是基于云技术的软件外包服务平台。利用云端软件开发协作平台 Coding.net 作为沟通和监管工具，通过智能匹配系统快速连接开发者与需求方，提供专业的项目顾问进行全程协助管理，加速软件开发交付模式转变，推动软件开发行业实现更高效的资源匹配。我在这个项目中参与了前后端的日常开发工作，技术栈主要有：Java，Spring MVC 框架，Mysql 数据库， Velocity 等。
